用户任务

帮我为今天的 Spark Jam 选择一个数据相关选题，并准备可以现场展示的产品故事。

主流程

  1. 导入团队资料、私有文档、agent 中间结论。
  2. 每条数据打上来源、置信度、权限、过期时间、敏感度。
  3. agent 只使用可追溯资料，冲突和隐私风险进入人工复核。
  4. 用户撤销、修正或确认记忆后再生成最终回答。
