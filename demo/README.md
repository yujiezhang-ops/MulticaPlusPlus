# Agent Memory Ledger Demo

Static Spark Jam prototype for the data-themed idea shortlist.

Open `agent-memory-ledger.html` directly in a browser, or serve the repository
root and visit `/demo/agent-memory-ledger.html`.

The page uses mock data from `src/agent-memory-ledger.js` and demonstrates:

- data source inventory
- quality scoring across source trust, freshness, privacy safety, and relevance
- policy decisions for usable, review-needed, and rejected agent memories
- conflict and permission alerts
- a compact pitch flow for the jam presentation
